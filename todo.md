# FinWise - Sistema de Gestão Financeira

## Funcionalidades Principais

- [x] Dashboard financeiro com visão geral de receitas, despesas, saldo mensal e gráficos interativos
- [x] Sistema de registro manual de transações (receitas e despesas) com categorização automática por IA
- [x] Integração com WhatsApp simulada para registro de transações via texto e áudio usando Speech-to-Text e LLM
- [x] Ferramenta de importação e categorização automática de extratos bancários (OFX/CSV) e faturas de cartão
- [x] Planejador de orçamento mensal com definição de limites por categoria e acompanhamento em tempo real
- [x] Quadro de metas financeiras (Vision Board) com rastreamento de progresso e tempo estimado
- [x] Rastreador de dívidas com status (em aberto, em negociação, finalizado) e controle de parcelas
- [x] Controle de múltiplas contas bancárias e cartões de crédito com saldos individuais
- [x] Sistema de acesso multi-usuário para membros da família compartilharem finanças
- [x] Personalização de tema com 8 esquemas de cores diferentes (Azul, Rosa, Amarelo, Azul Claro, Laranja, Lilás, Preto, Verde)
- [x] Sistema de notificações in-app para alertas de orçamento, metas e dívidas
- [x] Gerar insights e análises personalizadas sobre padrões de gastos, sugestões de economia e previsões financeiras usando IA

## Estrutura Técnica

- [x] Schema do banco de dados (transações, categorias, orçamentos, metas, dívidas, contas, cartões, famílias)
- [x] API tRPC com procedures para todas as funcionalidades
- [x] Interface em português brasileiro
- [x] Testes unitários com Vitest (25 testes passando)
